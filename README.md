# 📊 Sales Analytics Dashboard

## 📌 Overview

The Sales Performance Dashboard is an interactive business intelligence project developed to analyze sales performance and provide actionable insights. The dashboard enables users to monitor key performance indicators (KPIs), identify sales trends, evaluate regional performance, and make data-driven business decisions.

---

## ✨ Features

- Interactive Dashboard
- Sales Trend Analysis
- Monthly and Yearly Performance
- Category-wise Sales Analysis
- Regional Sales Analysis
- Profit Analysis
- KPI Cards
- Dynamic Filters

---

## 🛠️ Tools & Technologies

- Tableau
- Microsoft Excel
- SQL
- Data Visualization

---

## 📊 Dashboard Insights

- Total Sales
- Total Profit
- Total Orders
- Monthly Sales Trends
- Regional Performance
- Category Performance
- Top Performing Products
